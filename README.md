# 🎮 下班之道：Overtime War

**拒絕加班！打響你的下班戰爭！！** 🕒⚔️

**下班不是權利，而是一場世界級的戰爭！**

你不再是乖乖社畜，而是奮戰在辦公室戰場上的下班戰士！  
透過選擇不同角色（高層、中層、普通員工、Hea底員工），運用心機與策略，互相影響工時，

### ✨ 遊戲特色
- **回合制極智博弈**：輪流選擇角色，結合時間運用與心計權謀（你早走，我就要比你更早走！）
- 豐富的角色系統與連鎖副作用（高層壓制、Hea底反殺、中層壓力、基層怨氣）
- 多人即時燒腦對戰（適合辦公室同事一起玩）
- 緊張刺激的工時計算與辦公室時鐘機制  
- 一邊在辦公室996，一邊享受下班的終極爽感！
